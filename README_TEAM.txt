RICE Tester - Team Setup Instructions
=====================================

FIRST TIME SETUP:
1. Extract the ZIP file to your desired location
2. Run "SETUP_FIRST_TIME.bat" to install dependencies
3. Wait for installation to complete

DAILY USE:
- Run "python RICE_Tester.py" to start the application
- Or double-click RICE_Tester.py if Python is associated

REQUIRED DEPENDENCIES:
- Python 3.8 or higher
- All dependencies will be installed automatically by setup script

TROUBLESHOOTING:
- If responsive design looks wrong, ensure all dependencies are installed
- Run setup script again if needed
- Check that your screen resolution is supported (minimum 1024x768)

FEATURES:
- Responsive design adapts to different screen sizes
- Personal analytics dashboard
- Auto-update system
- GitHub integration
- Professional UI with Infor branding
